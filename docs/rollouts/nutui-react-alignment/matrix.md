# NutUI React Native Alignment Matrix

This is the seed matrix for the rollout. The full matrix should be regenerated by the inventory tooling in phase `00-baseline-contract`.

Status values:

- `complete`: target behavior is already implemented and documented.
- `partial`: implementation exists but API, visual behavior, tokens, docs, demo, or tests are incomplete.
- `missing`: target component does not exist in RN.
- `config-only`: current config mentions it, but no component directory exists.
- `risky`: implementation exists but RN behavior needs redesign or stronger verification.
- `unsupported`: target behavior is not suitable for RN without an approved substitute.
